# 安全性與合規性 🔒

**考試比重：25%**

這個領域涵蓋AWS雲端中的安全性與合規性概念，是CCP考試的重要組成部分。

## 主要學習內容

### AWS 共同責任模型
- AWS 負責「雲端的」安全性
- 客戶負責「雲端中的」安全性
- 服務分類：IaaS、PaaS、SaaS 的責任劃分

### IAM 身份與存取管理
- 使用者、群組、角色、政策
- 多重要素驗證 (MFA)
- 最小權限原則

### 合規性程式
- AWS Artifact 合規性文件
- 各種合規性框架 (SOC, PCI DSS, HIPAA)
- AWS Config 組態管理

---

> 💡 **學習重點**: 理解責任劃分和基本安全概念。
