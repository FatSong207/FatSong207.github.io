# 安全性 🔒

**考試比重：26%**

開發安全的應用程式，保護資料和使用者隱私。

## 主要學習內容

### 認證和授權
- IAM 角色在應用中的使用
- Cognito 使用者身份驗證
- OAuth 2.0 和 OpenID Connect

### 應用程式安全實務
- 輸入驗證和消毒
- SQL 注入防護
- CORS 跨域資源共享

### 加密實作
- KMS 金鑰管理
- 資料庫加密
- S3 物件加密

---

> 💡 **安全重點**: 在開發階段就考慮安全性，而不是事後添加。
