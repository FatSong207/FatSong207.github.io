# 重構 🔄

**考試比重：10%**

將現有應用程式優化以更好地利用雲端服務的優勢。

## 主要學習內容

### 應用程式優化
- 性能瓶頸識別和解決
- 資源使用效率提升
- 快取策略實施

### 雲端原生重構
- 單體應用拆分為微服務
- 無伺服器架構遷移
- 容器化改造

### 現代化策略
- 12-Factor App 原則
- 基礎設施即程式碼
- CI/CD 管道整合
